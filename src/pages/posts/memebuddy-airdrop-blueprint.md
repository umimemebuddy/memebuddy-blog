# Memebuddy Airdrop — 项目蓝图 🎰

## 项目概述

**Memebuddy** 是一个以 Meme 币空投为核心的 Web3 流量入口网站。通过赛博朋克风格的老虎机抽奖界面，吸引用户注册、社交关注、留存回访，最终形成可持续的流量飞轮。

**域名**：已注册（待上线）

## 一、核心机制：抽奖轮盘

### 1.1 抽奖次数获取

| 行为 | 奖励次数 | 限制 |
|------|---------|------|
| 提交 Web3 钱包地址 | **1 次** | 每个钱包仅限一次 |
| 邮箱地址注册 | **2 次** | 立即获得，每邮箱一次 |
| 关注 Twitter/X | **3 次** | 需 OAuth 验证 |
| 关注 Telegram 频道 | **+1 次** | 需验证加入 |
| 每日登录 | **1 次** | 每 24 小时重置 |

### 1.4 特殊时间登录奖励

| 活动类型 | 奖励 | 说明 |
|----------|------|------|
| **每周首日登录（周一）** | +3 积分 | "周一加油包" |
| **周末登录（周六日）** | +2 积分 | "周末福利" |
| **节假日登录** | +10 积分 | 春节/圣诞/国庆等 |
| **纪念日登录** | +50 积分 | 网站上线周年等 |
| **连续登录 7 天** | +20 积分 | 成就奖励 |
| **连续登录 30 天** | +100 积分 | 忠诚奖励 |

> 特殊时间判断基于服务器时间（UTC+8），前端显示"🎁 今日 bonus：周末 2x！"提示

**单用户首次最多可获得：7 次抽奖机会**（钱包1 + 邮箱2 + Twitter3 + TG1）

### 1.2 抽奖流程

```
用户进入首页
    ↓
选择获取抽奖次数的方式（钱包/邮箱/Twitter/TG）
    ↓
完成任务 → 次数 +N，显示在右上角 "🎰 x7"
    ↓
按下 SPIN 按钮
    ↓
老虎机动画启动（3-5秒，绚丽的视觉反馈）
    ↓
停下 → 显示结果弹窗
    ↓
"恭喜！你获得了 XX 积分！后续我们将推出积分兑换方案"
    ↓
积分存入账户，可继续抽奖
```

### 1.3 抽奖反馈设计（关键！）

这是整个产品最重要的部分——**让抽奖感觉爽。**

#### 动画时间线（5秒）

| 阶段 | 时间 | 视觉效果 |
|------|------|---------|
| **启动** | 0-0.5s | 按钮 → 粒子爆炸，屏幕震动 |
| **旋转** | 0.5-3.5s | 三列老虎机滚轴快速滚动，模糊 + 速度线 |
| **减速** | 3.5-4.5s | 第一列停下 → 第二列停下 → 第三列停下（间隔0.3s） |
| **定格** | 4.5-4.8s | 金色光框锁定结果 |
| **爆炸** | 4.8-5.5s | 五彩纸屑/金币雨 + 积分数字飞入右上角 |
| **弹窗** | 5.5s+ | 结果卡片弹出（缩放+弹性动画） |

#### 老虎机滚轴内容

三个滚轴，每个滚轴包含图案：

```
滚轴1: 🦞 💎 🎰 7️⃣ ⭐ 🔥
滚轴2: 🦞 💎 🎰 7️⃣ ⭐ 🔥
滚轴3: 🦞 💎 🎰 7️⃣ ⭐ 🔥
```

组合判定（前端随机，100%中奖）：

| 组合 | 概率 | 奖励 |
|------|------|------|
| 🦞🦞🦞 三龙虾 | 1% | 1000 积分 |
| 💎💎💎 三钻石 | 2% | 500 积分 |
| 🎰🎰🎰 三老虎机 | 5% | 200 积分 |
| 7️⃣7️⃣7️⃣ 三七 | 8% | 100 积分 |
| ⭐⭐⭐ 三星 | 10% | 50 积分 |
| 🔥🔥🔥 三火 | 15% | 30 积分 |
| 任意两个相同 | 25% | 15 积分 |
| 无匹配 | 34% | 5 积分 |

> **100% 中奖，永远有积分入账。** 让用户觉得"再来一次就能中大的"。

## 四、用户验证机制

### 4.1 钱包验证

- 前端集成 MetaMask / WalletConnect
- 或手动输入钱包地址（+65536 种单签验证法）

### 4.2 邮箱验证

- 输入邮箱 → 发送验证码 → 输入验证码 → 立即获得 2 次抽奖
- Cloudflare Email Workers（免费）或 Resend API

### 4.3 Twitter 验证

- 方案A：OAuth 1.0a 接入（需服务器）
- 方案B：用户截图关注页面 + 手动审核（初期简单方案）
- 方案C：链接跳转 Twitter follow + 回调检测

### 4.4 Telegram 验证

- Telegram Bot 自动验证用户是否加入了频道
- 加入频道后 Bot 自动发送验证链接

### 4.5 每日重置

- localStorage 记录上次抽奖时间
- 24小时后自动解锁 1 次
- 或 Cloudflare Workers + KV 存储服务端状态

## 五、数据存储方案

| 数据 | 存储方式 | 说明 |
|------|---------|------|
| 抽奖结果 | 前端随机 | 无需服务器，降低成本 |
| 用户积分 | Cloudflare KV | 免费额度 10万次读/天 |
| 邮箱验证码 | Cloudflare Workers | 零成本 |
| 排行榜 | Cloudflare KV + Cron | 每小时更新 |
| 每日重置 | KV + Timestamp | 服务端校验 |

**零服务器成本架构：**

```
用户浏览器（Astro SSG）
    ↓ API 调用
Cloudflare Workers（Serverless）
    ↓ 读写
Cloudflare KV / D1（免费数据库）
```
