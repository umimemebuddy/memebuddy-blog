# FOMO Rider 开发日志

> 一款关于加密货币交易员在K线上滑行的无尽跑酷游戏。

## 核心概念

FOMO Rider 的灵感很简单：**一个交易员坐在K线上，随着价格起伏滑行**。

这不是模拟交易，而是体验交易。玩家不需要分析图表，只需要感受那种「按住买入、松开卖出」的原始冲动。K线向上时你在赚钱，K线向下时你在亏钱——就这么简单。

## 操控设计：极简主义

游戏只有一个动作：

- **按住** = 买入（全仓梭哈）
- **松开** = 卖出（落袋为安）

没有复杂的订单类型，没有杠杆倍数选择，没有止盈止损设置。我们剥离了一切「专业交易」的表象，只留下最纯粹的 FOMO（Fear Of Missing Out）。

## 视觉风格：工业美学

### 色彩系统

- **背景**：OLED 纯黑 `#000000`
- **上涨**：霓虹绿 `#00ff88` + 发光效果
- **下跌**：警示红 `#ff4444` + 发光效果
- **财富**：金色 `#ffd700`（达成 $1M 时触发）

拒绝圆角、拒绝渐变、拒绝阴影。只有硬边几何和高对比度。

### 字体选择

- **Space Grotesk**：余额数字，科技感
- **JetBrains Mono**：标签文字，技术感

### 主角设计

交易员是一只**龙虾**——不是随便选的。

在中文加密社区，「虾」是散户的自嘲符号。我们给这只虾穿上了西装，让它坐在K线最前端，实时跟随价格起伏。买入时它发光变大，亏钱时它恐慌抖动，暴富时它变成皇冠。

## 游戏机制

### K线生成

价格基于随机游走算法，波动率随时间递增。前期相对平稳，后期疯狂震荡——就像真实的加密市场。

## 积分系统

- **Trades**：交易次数
- **Best**：最佳单笔盈利
- **Combo**：连续盈利次数

连击 x10 时屏幕中央会显示巨大的紫色「x10」，x50 时则是「神之手」。

## 技术栈

纯静态三文件：

```
index.html  (8588 bytes)
```

- 无框架
- 无构建工具
- 无依赖
- Canvas API 绘制 K线
- SVG 绘制角色
- CSS 动画处理特效

这就是我们的开发哲学：**极简技术栈 + 快速迭代**。
